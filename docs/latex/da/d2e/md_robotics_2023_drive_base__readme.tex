The documentation pages and visuals are created by doxygen. The doxygen configuration is almost completely default, its file is \char`\"{}\+Doxygen\char`\"{} \href{https://www.doxygen.nl}{\texttt{ learn more about doxygen}}

When updating code, comments, or anything that should be in documentation, run {\ttfamily doxygen} before commiting. Install Doxygen on mac with \href{https://brew.sh}{\texttt{ Brew}}\+: {\ttfamily brew install doxygen}

If you want to change the doxygen configuration, use the doxygen gui, \href{https://www.doxygen.nl/download.html}{\texttt{ doxywizard}}. Make sure you save the outputted doxygen config in the Doxygen file.\hypertarget{md_robotics_2023_drive_base__readme_autotoc_md6}{}\doxysubsection{How the drive base works}\label{md_robotics_2023_drive_base__readme_autotoc_md6}
The \char`\"{}heart\char`\"{} of the robot is the roborio, which uses wpilib for instructions. There are a couple \href{https://docs.wpilib.org/en/stable/docs/software/vscode-overview/creating-robot-program.html}{\texttt{ base robot templates}}, but we are using the {\ttfamily Timed\+Robot}.

If you want to understand wpilib, start \href{https://docs.wpilib.org/en/stable/docs/zero-to-robot/introduction.html}{\texttt{ here}}. After that, you will need the \href{https://github.wpilib.org/allwpilib/docs/release/java/}{\texttt{ Java library documentation}}.

Then, learn about our \href{https://cchs-first-robotics.github.io/2023DriveBase/html/annotated.html}{\texttt{ drivebase code}}.\hypertarget{md_robotics_2023_drive_base__readme_autotoc_md7}{}\doxysubsection{Motors and components}\label{md_robotics_2023_drive_base__readme_autotoc_md7}
Motors/motor controllers and other components of the robot will need other java libraries, and have other docs aswell.

We are using \href{https://codedocs.revrobotics.com/java/index.html}{\texttt{ Rev Robotics Spark\+Max}} and \href{https://api.ctr-electronics.com/phoenix/release/java/com/ctre/phoenix/motorcontrol/package-summary.html}{\texttt{ ctre Phoenix Talon and Victor}} motor controllers (links are to their libraries).

The robot also uses a \href{https://www.kauailabs.com/public_files/navx-mxp/apidocs/java/com/kauailabs/navx/frc/package-summary.html}{\texttt{ NavX Gyro}}. The gyro can get lots of information about the robot\textquotesingle{}s orientation, position, etc.

When adding a new component, look up instructions on how to install it, and then update this doc with its library\textquotesingle{}s documenation.
