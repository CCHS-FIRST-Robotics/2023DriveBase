drive the robot using controller inputs

Left analog stick controls translation Right analog stick controls rotation (move it right -> rotate clockwise, move it left -> rotate counterclockwise)

positive is right/up

Parameters
leftAnalogXthe x position of the left analog stick; range: [-1, 1]
leftAnalogYthe y position of the left analog stick; range: [-1, 1]
rightAnalogXthe x position of the right analog stick; range: [-1, 1]
rightAnalogYthe y position of the right analog stick; range: [-1, 1]

Debug mode: 

When you press the left stick down (left stick button) the robot enters

debug mode (bool debugMode), and the robot will only move power one wheel at a time. Toggle through these wheels by pressing the left stick button more (FL -> FR -> BL -> BR) then, after BR, it will leave debug mode and go back into all wheel drive

Add deadzone (stop all movement when input is under a certain amount) Compare joystick distance from normal position (0)
